# Scaffold Pi Extension Project

## Summary

Initialize the `spec-teams-extension` repository with the baseline structure for developing a Pi coding agent extension.

## Motivation

The repository is empty aside from OpenSpec and Pi skill scaffolding. We need the foundational files to begin developing a Pi extension:

- `package.json` with Pi package metadata and peer dependencies
- `extensions/index.ts` — a minimal extension entry point
- `tsconfig.json` for IDE support
- Updated OpenSpec config with project context

## Scope

- Scaffold the minimal project skeleton
- Not implementing any extension features yet — just the bones

## Non-goals

- No tools, commands, events, or UI in this change
- No tests framework setup (can be added later)
